# 25. 迷你播放器 (SDL2 显示)

## 等价命令
```bash
ffplay input.mp4
```

## 这条命令做了什么
解码视频并用 SDL2 窗口实时显示, 这是一个最简单的视频播放器。

## 新增依赖
```
SDL2 - 跨平台多媒体库, 用于创建窗口和渲染画面
```

## 你将学到
- SDL2 窗口创建和纹理渲染
- 解码帧率控制 (按 pts 控制显示节奏)
- 把前面所有知识串成一个可交互的应用

## 编译运行
```bash
# 需要安装 SDL2: brew install sdl2 (macOS)
./25_mini_player input.mp4
```
