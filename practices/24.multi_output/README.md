# 24. 一个输入, 多个输出 (多分辨率)

## 等价命令
```bash
ffmpeg -i input.mp4 \
  -c:v libx264 -s 1920x1080 hd.mp4 \
  -c:v libx264 -s 640x480 sd.mp4
```

## 这条命令做了什么
一次解码, 同时编码为两个不同分辨率的输出。
这是直播/点播中常见的"多码率转码"场景。

## 你将学到
- 一个解码帧同时送给多个编码器
- 管理多个输出上下文
- 实际应用: 自适应码率 (ABR) 转码

## 编译运行
```bash
./24_multi_output input.mp4 hd.mp4 1920x1080 sd.mp4 640x480
```
